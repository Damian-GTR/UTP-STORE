FLUJO DEL BACKEND - UTP-STORE
============================

Este documento explica cómo funciona el backend para cargar productos dinámicamente desde PostgreSQL.


1. ESTRUCTURA DE ARCHIVOS
--------------------------
- db.js: Configuración de conexión a PostgreSQL
- server.js: Servidor Express con rutas API
- js/productos.js: JavaScript para Productos.html
- js/detalles.js: JavaScript para Detalles.html


2. db.js - CONEXIÓN A LA BASE DE DATOS
---------------------------------------
const { Pool } = require("pg");

const pool = new Pool({
    user: "postgres",
    host: "localhost",
    database: "utp_store",
    password: "andree2020",
    port: 5432,
});

module.exports = pool;

EXPLICACIÓN:
- Crea un "pool" de conexiones a PostgreSQL
- Configura credenciales de acceso
- Exporta el pool para usarlo en server.js


3. server.js - SERVIDOR EXPRESS
--------------------------------
const express = require("express");
const cors = require("cors");
const pool = require("./db");

const app = express();

app.use(cors());
app.use(express.json());

EXPLICACIÓN:
- express: Framework para crear servidor web
- cors: Permite que el frontend acceda al backend (evita bloqueos de seguridad)
- pool: Importa la conexión a PostgreSQL
- app.use(cors()): Habilita CORS para permitir peticiones desde el navegador


4. RUTAS DEL SERVIDOR
----------------------

RUTA 1: Obtener todos los productos
------------------------------------
app.get("/productos", async (req, res) => {
    try {
        const resultado = await pool.query("SELECT * FROM productos");
        res.json(resultado.rows);
    } catch (error) {
        console.error(error);
        res.status(500).send("Error al obtener los productos");
    }
});

EXPLICACIÓN:
- GET /productos: Endpoint para obtener todos los productos
- pool.query("SELECT * FROM productos"): Ejecuta consulta SQL
- res.json(resultado.rows): Devuelve los productos en formato JSON
- Si hay error, devuelve status 500


RUTA 2: Obtener un producto por ID
-----------------------------------
app.get("/productos/:id", async (req, res) => {
    try {
        const { id } = req.params;
        const resultado = await pool.query("SELECT * FROM productos WHERE id = $1", [id]);
        if (resultado.rows.length === 0) {
            res.status(404).send("Producto no encontrado");
        } else {
            res.json(resultado.rows[0]);
        }
    } catch (error) {
        console.error(error);
        res.status(500).send("Error al obtener el producto");
    }
});

EXPLICACIÓN:
- GET /productos/:id: Endpoint para obtener un producto específico
- req.params.id: Extrae el ID de la URL (ej: /productos/1 → id = "1")
- $1: Parámetro en la consulta SQL (previene inyección SQL)
- Si no encuentra el producto, devuelve 404
- Si lo encuentra, devuelve el producto en JSON

const PORT = 3000;

8. FLUJO COMPLETO - PRODUCTOS.HTML
----------------------------------
1. Usuario abre Productos.html
2. El navegador carga js/productos.js
3. productos.js hace fetch a http://localhost:3000/productos
4. server.js recibe la petición GET /productos
5. server.js consulta PostgreSQL: SELECT * FROM productos
6. PostgreSQL devuelve los datos
7. server.js devuelve JSON con los productos
8. productos.js recibe el JSON
9. productos.js filtra por categoría
10. productos.js genera HTML dinámico con las tarjetas
11. productos.js inyecta el HTML en los contenedores (contenedor-cuadernos, etc.)
12. El navegador muestra los productos

15. LOCALSTORAGE - CARRITO DE COMPRAS
------------------------------------
localStorage es una "base de datos" en el navegador que persiste datos.

GUARDAR DATOS:
localStorage.setItem('carrito', JSON.stringify(carrito));

LEER DATOS:
let carrito = JSON.parse(localStorage.getItem('carrito')) || [];

COMUNICACIÓN ENTRE ARCHIVOS:
- detalles.js: Guarda productos cuando haces clic en "Agregar al carrito"
- carrito.js: Lee localStorage cuando abres Carrito.html
- Los datos persisten aunque cierres el navegador


16. js/detalles.js - AGREGAR AL CARRITO
----------------------------------------
function agregarAlCarrito(id, nombre, precio, imagen) {
    let carrito = JSON.parse(localStorage.getItem('carrito')) || [];

    const productoExistente = carrito.find(p => p.id === id);

    if (productoExistente) {
        productoExistente.cantidad += 1;
    } else {
        carrito.push({
            id: id,
            nombre: nombre,
            precio: precio,
            imagen: imagen,
            cantidad: 1
        });
    }

    localStorage.setItem('carrito', JSON.stringify(carrito));
    alert('Producto agregado al carrito');
}

EXPLICACIÓN:
- Lee el carrito actual del localStorage
- Si el producto ya existe → incrementa cantidad
- Si no existe → agrega nuevo producto con cantidad = 1
- Guarda el carrito actualizado en localStorage


17. js/carrito.js - MOSTRAR CARRITO
------------------------------------
function mostrarCarrito() {
    if (carrito.length === 0) {
        mensajeCarritoVacio.classList.remove('d-none');
        resumenCarrito.classList.add('d-none');
        return;
    }

    mensajeCarritoVacio.classList.add('d-none');
    resumenCarrito.classList.remove('d-none');

    carrito.forEach((producto, index) => {
        contenedorCarrito.innerHTML += `<div class="card">...</div>`;
    });

    calcularTotal();
}

EXPLICACIÓN:
- Si carrito vacío → muestra mensaje, oculta resumen
- Si carrito con productos → oculta mensaje, muestra resumen
- Inyecta tarjetas HTML dinámicamente para cada producto
- Calcula subtotal, IGV y total


18. js/carrito.js - MODIFICAR CANTIDAD
--------------------------------------
function modificarCantidad(index, cambio) {
    carrito[index].cantidad += cambio;

    if (carrito[index].cantidad <= 0) {
        carrito.splice(index, 1);
    }

    localStorage.setItem('carrito', JSON.stringify(carrito));
    mostrarCarrito();
}

EXPLICACIÓN:
- Incrementa o decrementa cantidad del producto
- Si cantidad <= 0 → elimina producto del carrito
- Guarda cambios en localStorage
- Actualiza la vista

21. js/carrito.js - CALCULAR TOTAL
-----------------------------------
function calcularTotal() {
    const subtotal = carrito.reduce((total, producto) => {
        return total + (producto.precio * producto.cantidad);
    }, 0);

    const igv = subtotal * 0.18;
    const total = subtotal + igv;

    document.getElementById('subtotal').textContent = `$${subtotal.toFixed(2)}`;
    document.getElementById('igv').textContent = `$${igv.toFixed(2)}`;
    document.getElementById('total').textContent = `$${total.toFixed(2)}`;
}

EXPLICACIÓN:
- reduce(): Suma (precio × cantidad) de todos los productos
- Calcula IGV (18% del subtotal)
- Total = subtotal + IGV
- Actualiza los elementos HTML con los valores calculados
